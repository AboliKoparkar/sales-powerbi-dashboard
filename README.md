# 📊 Sales Performance Dashboard (Power BI)

## 🔍 Project Overview

This Power BI dashboard analyzes sales data to generate business insights.

## 📁 Dataset

* Orders data including category, sales, profit, and payment mode.

## 📊 Key Insights

* Electronics category generated highest revenue
* Maharashtra has highest sales
* COD is most used payment method
* Profit varies across sub-categories

## 🛠 Tools Used

* Power BI
* Excel

## 🚀 Learnings

* Data cleaning using Power Query
* Data modeling (relationships)
* Dashboard building
